#AgroSustentável
🎯 Objetivo do Site
O site foi desenvolvido com o propósito de engajar e conscientizar o público sobre o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”. Seus objetivos específicos são:

Conectar Produção e Preservação: Mostrar de forma prática que o aumento da produtividade agrícola não precisa caminhar separado da conservação ambiental.

Destacar a Agricultura Familiar: Utilizar dados oficiais do Censo Agropecuário do IBGE para dar protagonismo aos pequenos produtores, evidenciando que eles produzem a maior parte dos alimentos que consumimos enquanto protegem os recursos naturais.

Educar de Forma Lúdica: Proporcionar uma experiência interativa e visualmente leve (através de recursos dinâmicos em JavaScript), facilitando o entendimento de técnicas sustentáveis como o Sistema Agroflorestal e a rotação de culturas.
omo o projeto possui regras estritas quanto ao uso de tecnologias (sendo permitido apenas HTML, CSS e JavaScript puros), os programas sugeridos e utilizados para o desenvolvimento são focados em edição de código, design e organização:

Visual Studio Code (VS Code): O editor de código (IDE) principal utilizado para escrever e estruturar o HTML, estilizar com CSS e programar as funções em JavaScript.

Navegador Web (Google Chrome / Mozilla Firefox): Utilizado para testar o site em tempo real, inspecionar elementos e garantir que o layout seja totalmente responsivo (adaptável a celulares e computadores).

Figma ou Canva (Opcional - Design): Utilizados na etapa inicial de planejamento para a escolha da paleta de cores lúdicas, fontes e esboço do layout (Wireframe).

Git / GitHub (Opcional - Controle de Versão): Utilizado para salvar o progresso do código com segurança e garantir que nenhuma versão com erros fosse perdida.
